# Welcome to Marka Editor

Open source markdown editor for anyone who need fast, distraction-free markdown editing.

## Getting Started

### Writing Markdown

Start typing in the editor to the left. The editor supports all standard markdown syntax:

- **Bold text**: `**text**` or `__text__`
- _Italic text_: `*text*` or `_text_`
- `Inline code`: wrap with backticks
- [Links](https://example.com): `[text](url)`

### Using the Toolbar

The toolbar at the top provides quick access to formatting commands:

- **Heading (H1, H2, H3)**: Insert markdown headers
- **Bold, Italic, Strikethrough**: Quick formatting
- **Code Block**: Insert multi-line code
- **Blockquote**: Add quoted text
- **List**: Create unordered or ordered lists
- **Link, Image**: Insert media
- **Undo/Redo**: Navigate edit history

### Preview Mode

Use the **toggle button** in the toolbar to switch between syntax (editor) mode and formatted (preview) mode. Preview shows your formatted markdown in real-time.

## Tips

- **File Size**: Check the status bar (bottom right) to see your document's character count and file size
- **Keyboard Shortcuts**: Use Ctrl+Z to undo and Ctrl+Y to redo
- **Line Count**: The status bar shows your document's line count for quick reference
- **Markdown Syntax**: If you're new to markdown, the preview mode shows you exactly how your text will look

## Markdown Syntax Reference

```markdown
# Heading 1

## Heading 2

### Heading 3

**Bold** and _italic_ text

- Bullet point
- Another point

1. Numbered item
2. Second item

[Link text](https://example.com)

![Alt text](image-url)

> This is a blockquote

`inline code`

\`\`\`
code block
\`\`\`
```
